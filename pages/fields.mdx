Widgets can be added through the slimplateProvider

```jsx
 <SlimplateProvider
    widgets={widgets}
    corsProxy={window?.slimplate?.corsProxy}
    backendURL={window?.slimplate?.backendURL}
    monoProject={window?.slimplate?.project}
  >
    <App />
  </SlimplateProvider>
```

We also offer a few default widgets which you can import.

```jsx
import {widgets } from '@slimplate/react-flowbite-github'
```

AdminEdit then uses widgets to create form inputs for each field type in the collection.

```jsx
 "collections": {
    "page": {
      "title": "Page",
      "files": "/pages/**/*.mdx",
      "fields": [
        {
          "name": "title",
          "label": "Title",
          "validators": [
            "required"
          ]
        },
        {
          "name": "content",
          "label": "Body",
          "type": "richtext",
          "validators": [
            "required"
          ]
        }
      ]
    }
  }
```
