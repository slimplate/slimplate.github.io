Widgets can be added through the slimplateProvider, by default we offer a few which you can import.

```jsx
import {widgets } from '@slimplate/react-flowbite-github'

 <SlimplateProvider
    widgets={widgets}
    corsProxy={window?.slimplate?.corsProxy}
    backendURL={window?.slimplate?.backendURL}
    monoProject={window?.slimplate?.project}
  >
    <App />
  </SlimplateProvider>
```

The [AdminEdit](/admin-views#admin-edit) component then uses widgets to create form inputs for each field type in the articles slimplate.json collection obj. Important to note that fields without a type field will be treated as a string.

```jsx
 "collections": {
    "page": {
      "title": "Page",
      "files": "/pages/**/*.mdx",
      "fields": [
        {
          "name": "title",
          "label": "Title",
          "validators": [
            "required"
          ]
        },
        {
          "name": "content",
          "label": "Body",
          "type": "richtext",
          "validators": [
            "required"
          ]
        }
      ]
    }
  }
```

Here is a list of types we offer by default:

*   color
*   date
*   string
*   number
*   richtext
